Valorant Rank Tracker

Project Type: Python, HTML, CSS, Flask, API

Purpose

While there are a variety of custom apps that provide player metrics for Valorant, I attempt to provide players the ability to view their historical metrics to provide insight into player behavior, preferences as well as potential forecasting.

A walkthrough

I used the RIOT API which provides player in-game information for RIOT titles. Front end queries are done through Python and the RIOT API.
